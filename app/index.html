DevOps Demo App

This application is a small PHP + MySQL website used to practice Docker, CI/CD with Jenkins, and operational scripting on an AWS EC2 instance.

Containerisation
Runs as a Dockerised Apache + PHP service with a separate MySQL container.
CI/CD
Jenkins pipeline builds images, runs lint checks, and executes smoke tests.
Operations
Health check, backup, and smoke test scripts for day‑to‑day operations.
